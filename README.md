# Rule-Based Chatbot

## Description :
A simple conversational agent that responds to user inputs based on predefined rules. This project demonstrates basic Natural Language Processing (NLP) concepts using conditional logic.

## Logic Flow :
The bot looks for specific keywords in the user's input:
* **Greetings:** Responds to "hello" or "hi".
* **Status:** Responds to "how are you".
* **Identity:** Responds to "name".
* **Exit:** Closes the program when "bye" is typed.

## How to Run :
1. Run: `python chatbot.py`
2. Type your message in the terminal and press Enter.
3. Type 'bye' to end the conversation.
